# Clutter

## Installation

### Using GO

```
go install github.com/cluttercode/clutter/cmd/clutter
```

### From Source

```
make install
```

### Using gobinaries

```
$ curl -sf https://gobinaries.com/cluttercode/clutter/cmd/clutter@latest | sh
```

### Prebuilt binaries

From [releases](https://github.com/cluttercode/clutter/releases).
